# ARP_Spoofing_Detection_with_Wireshark
This repository contains the resources and findings from my end-of-course project for the City of London Cybersecurity Bootcamp. The project demonstrates the simulation and detection of ARP spoofing attacks within a controlled virtual lab environment using Kali Linux, Windows 7, and Wireshark.

Please note: Some IP and MAC addreses have been redacted or changed for privacy reasons.
